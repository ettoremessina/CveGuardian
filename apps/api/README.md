# CveGuardian API

The backend service for CveGuardian.

## Features
- **Hono** based API server.
- **Scanning Engine**: Orchestrates repository cloning and `DepScanity` execution.
- **Project Management**: CRUD operations for projects.
- **Stats & Dashboard**: Aggregated limits and alerts.
- **Search & Filter**: Powerful filtering for CVEs and impacted projects.
